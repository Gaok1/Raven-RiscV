RAVEN - atividade guiada

Estrutura da pasta
- programas/: arquivos .s a importar no Editor
- config-global/: arquivos .rcfg para a aba Config
- config-pipeline/: arquivos .pcfg para a aba Pipeline
- config-cache/: arquivos .fcache para a aba Cache

Programas
- programas/D101.s: pipeline com instrucoes independentes
- programas/D102.s: pipeline com cadeia de dependencias RAW
- programas/D201.s: load-use hazard
- programas/D202.s: flush por redirecionamento de controle
- programas/D301.s: streaming / AMAT / LRU vs FIFO
- programas/D302.s: thrashing por associatividade insuficiente
- programas/D401.s: comparacao entre tipos I, R, M e S
- programas/D501.s: multi-core / registradores independentes

Perfis gerais (.rcfg)
- config-global/R100.rcfg: 1 hart em foco, cache desabilitada, pipeline habilitado por padrao, latencias-base de 1 ciclo
- config-global/R300.rcfg: 1 hart em foco, cache habilitada, pipeline desligado, latencias-base de 1 ciclo
- config-global/R500.rcfg: escopo All Harts, cache desabilitada, pipeline desligado, 2 cores, latencias-base de 1 ciclo

Perfis de pipeline (.pcfg)
- config-pipeline/P100.pcfg: pipeline habilitado
- config-pipeline/P101.pcfg: pipeline desabilitado
- config-pipeline/P102.pcfg: pipeline habilitado, forwarding desabilitado
- config-pipeline/P103.pcfg: pipeline habilitado, forwarding ligado, preditor BTFNT

Perfis de cache (.fcache)
- config-cache/C311.fcache: D3/Q1 Config A (256 B, 2-way, hit latency 1)
- config-cache/C312.fcache: D3/Q1 Config B (1 KB, 2-way, hit latency 4)
- config-cache/C321.fcache: D3/Q2 streaming com LRU
- config-cache/C322.fcache: D3/Q2 streaming com FIFO
- config-cache/C331.fcache: D3/Q3 4 sets, 2 ways, line size 16
- config-cache/C332.fcache: D3/Q3 4 sets, 8 ways, line size 16

Mapeamento por dominio
- D1: use `programas/D101.s` com `config-pipeline/P100.pcfg` para D1-01; use `programas/D102.s` com `config-pipeline/P102.pcfg` para D1-02.
- D2: para `D201.s`, importe `config-global/R100.rcfg` na aba Config e `config-pipeline/P103.pcfg` na aba Pipeline; para `D202.s`, mantenha `config-pipeline/P100.pcfg`; abra o programa correspondente no Editor.
- D3/Q1: importe `config-global/R300.rcfg` na aba Config; mantenha `config-pipeline/P101.pcfg` na aba Pipeline; importe `config-cache/C311.fcache` ou `config-cache/C312.fcache` na aba Cache; abra `programas/D301b.s` no Editor.
- D3/Q2: importe `config-global/R300.rcfg` na aba Config; mantenha `config-pipeline/P101.pcfg` na aba Pipeline; importe `config-cache/C321.fcache` ou `config-cache/C322.fcache` na aba Cache; abra `programas/D301.s` no Editor.
- D3/Q3: importe `config-global/R300.rcfg` na aba Config; mantenha `config-pipeline/P101.pcfg` na aba Pipeline; importe `config-cache/C331.fcache` ou `config-cache/C332.fcache` na aba Cache; abra `programas/D302.s` no Editor.
- D4: importe `config-global/R100.rcfg` na aba Config; importe `config-pipeline/P101.pcfg` na aba Pipeline; abra `programas/D401.s` no Editor.
- D5: importe `config-global/R500.rcfg` na aba Config; mantenha `config-pipeline/P101.pcfg` na aba Pipeline; abra `programas/D501.s` no Editor.
- D6: importe `config-global/R100.rcfg` na aba Config; use `config-pipeline/P101.pcfg` para a medicao sem pipeline; troque apenas para `config-pipeline/P100.pcfg` na medicao com pipeline; reutilize `programas/D102.s`.
